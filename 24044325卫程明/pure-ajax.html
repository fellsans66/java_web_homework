纯 Ajax（XMLHttpRequest） 异步

使用浏览器原生 XMLHttpRequest 对象发起 HTTP 请求。 第三个参数 true 表示异步模式， 发送请求后不会阻塞页面，通过 onreadystatechange 回调监听响应。

点击按钮查看结果...
XMLHttpRequest.readyState 状态说明：
0 - UNSENT 请求未初始化（open() 未调用）
1 - OPENED open() 已调用
2 - HEADERS_RECEIVED 已收到响应头
3 - LOADING 响应体下载中
4 - DONE 请求完成（在此状态处理结果）
关键点： Ajax = Asynchronous JavaScript and XML。 XMLHttpRequest 第三个参数传 true 表示异步 —— 请求发出后立即返回， 不阻塞 UI 线程，数据返回后通过回调函数处理。如果传 false 则变成同步请求，会阻塞页面直到响应完成（不推荐使用）。
