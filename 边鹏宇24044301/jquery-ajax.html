jQuery Ajax 异步

jQuery 提供了 $.ajax()、$.get()、$.post() 等便捷方法， 自动处理 JSON 解析、跨浏览器兼容、错误回调等，极大简化 Ajax 编码。

点击按钮查看结果...
方法 说明 典型场景
$.get(url, data, callback) 简化 GET 请求 获取数据
$.post(url, data, callback) 简化 POST 请求 提交表单 / 创建资源
$.getJSON(url, data, callback) GET 请求并自动解析 JSON 获取 JSON 数据
$.ajax({ options }) 完全自定义 Ajax 请求 需要精细控制时
jQuery Ajax 优势： $.ajax() 支持丰富的配置项：timeout 超时控制、beforeSend 请求前回调、 complete 完成回调（无论成功失败）、dataType 自动数据类型转换。 还支持 Promise 风格的 .done()、.fail()、.always() 链式调用。
